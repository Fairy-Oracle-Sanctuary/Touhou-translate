## 🛠️ スクリプトとツールの紹介

このリポジトリは、翻訳、整理、公開のワークフロー効率を向上させるために設計された、一連の自主開発補助スクリプトとツールを提供します。

### Fairy-Kekkai-Workshop

`Fairy-Kekkai-Workshop` は、当プロジェクトが自主開発した統合Pythonツールであり、以下の核心機能を備えています：

- **プロジェクト管理**：様々な翻訳プロジェクトの進捗、リソース、メタデータを簡単に管理します。
- **リソースダウンロード**：プレイリストからの一括取得をサポートする、内蔵YouTube動画ダウンローダー。
- **リソース整理**：プロジェクト関連の画像、オーディオなどのアセットファイルを自動的に分類・整理します。

### Tools ディレクトリ

`tools` ディレクトリには、いくつかの実用的なユーティリティが含まれており、現在以下を提供しています：

- **ワンクリックプレイリストリソースダウンロード**：指定されたプレイリスト内のすべての動画リソースを素早く取得します。
- その他の補助機能。これらは将来、主ツール `Fairy-Kekkai-Workshop` に統合される予定です。

### 🚧 近日公開予定の機能

- **強化されたプロジェクト管理**：プロジェクト管理のための、より直感的で効率的なインターフェースと操作性。
- **ハードコード字幕抽出**：OCR技術を利用して、動画からハードコードされた字幕テキストを迅速に認識・抽出します。
- **スマート初稿タイミング生成**：音声認識に基づいて字幕の初期タイミングを自動生成し、初期のタイミング作業に必要な人手を大幅に削減します。
